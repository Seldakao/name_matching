This is a python file for WBAA interview task (2018). It is written with Python 3.6

In command prompt/ terminal(Mac OS), go to the appropriate directory where “WBAA_task.py” is located. 

Run Python 
Within Python environment, run  “ from WBAA_task import *”
Run  name_matching(path of G, path of Test), where path of G is a string of path of ground truth table, i.e. ‘user/A/B/C/G.csv’, and path of Test is the same for Test table
The function name_matching will save the prediction in a table called ‘Matching_table.csv’ under the directory
